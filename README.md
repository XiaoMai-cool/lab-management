## 功能模块

### 界面设计原则

**管理与使用完全分离：**
- 底部导航只显示使用功能：首页 | 物资 | 药品 | 报销 | 我的
- 管理功能通过"我的"页面的"管理功能"入口访问
- 老师/专人在管理模式下可直接操作，不需要走审批流程

### 登录页（无需登录即可查看）
- 公告通知（点击展开详情，紧急/重要/普通分级）
- 药品补货提醒（🔴即将用完 / 🟡已下单，只展示不可操作）
- 今日值日（实验室+办公室）+ 本周排班
- 日常须知（实验室/办公室分区）

### 耗材管理（使用功能）
- 物资总览：四分类，搜索
- 申领物资：提交申请 → 耗材专人（王子寒）审批发放
- 借用耗材：借用非一次性耗材 → 归还/报遗失
- 我的申领 / 我的借用：查看状态

### 药品管理（使用功能）
- 药品总览：120种药品，按编号分区（A/B/C/D/上），按类型分类
- 药品信息：名称、编号、分子式、CAS号、GHS安全标签
- **即将用完上报**：一键上报，无需备注，防重复
- 药品申购：提交申购 → 药品专人（宋艳芳）审批

### 采购审批 + 报销（三步流程）
```
第一步：学生提交采购审批 → 选老师审批 → 批准/拒绝
第二步：学生采购后提交报销 → 李健楠/大导审批
第三步：报销通过后自动分流 → 药品类→宋艳芳统计 / 非药品→王子寒统计
```
- 报销类别：个人药品、外送检测、设备配件、加工定制、办公打印、差旅费、邮寄快递、其他
- 文件上传：购买截图、发票、检测报告、资质证书（Supabase Storage）
- 报销统计：按月/按人汇总，导出CSV

### 药品补货管理（药品专人）
- 查看"即将用完"上报列表
- 标记"已下单"（填预计送达日期）
- 标记"已送达"（可跳步直接标记）
- 自动送达：到预计日期自动标记
- 可选更新库存数量

### 其他功能
- 制度文档：在线查看（4篇已录入）
- 值日排班：实验室每人一天/4周轮换 + 办公室每月一换
- 公告管理：发布/编辑/删除

### 管理后台（通过"我的"→"管理功能"进入）
- 系统管理、公告管理、人员管理、数据导出（大导）
- 耗材审批、耗材管理、借用管理、报销统计（耗材专人）
- 药品补货、药品管理、供应商管理、报销统计（药品专人）
- 采购审批（老师）
- 报销审批（李健楠/大导）
- 值日管理（日常管理）

## 项目结构

```
src/
├── components/           # 通用组件（SubNav, Layout, Modal, Card...）
├── contexts/AuthContext   # 认证（角色权限helper）
├── pages/
│   ├── Dashboard          # 首页（角色定制widget）
│   ├── auth/LoginPage     # 登录页（公告/补货/值日/须知）
│   ├── supplies/          # 耗材（7页：总览/申领/借用/归还/审批/管理）
│   ├── reagents/          # 药品（6页：总览/详情/编辑/申购/供应商/补货管理）
│   ├── reimbursements/    # 报销（4页：列表/申请/审批/统计）
│   ├── purchase-approvals/# 采购审批（3页：申请/列表/审批）
│   ├── documents/         # 制度文档（3页）
│   ├── duty/              # 值日（1页）
│   ├── profile/           # 个人中心（含管理入口）
│   └── admin/             # 管理后台（5页）
└── App.tsx                # 路由配置
```
