## 功能模块

### 界面设计原则

**管理与使用分离，管理按职责分组：**
- 底部导航：首页 | 物资 | 药品 | 采购 | 我的
- 管理面板按职责分组（彩色标题区分）：
  - 采购审批（绿色，教师独立入口）
  - 报销管理（黄色：报销审批、报销统计）
  - 耗材管理（蓝色：申领审批、库存管理、物资追踪、入库登记）
  - 药品管理（紫色：药品库存、补货管理、入库登记）
  - 系统管理（灰色：公告、人员、日常须知、数据导出）

### 登录页（无需登录即可查看）
- 公告通知（管理员可选择哪些公告在登录页展示，点击展开详情）

### 首页 Dashboard
- 待办事项（管理角色看到待审批数量、今日归还提醒等）
- 公告栏（支持图片展示和文件下载，查看全部链接）
- 快捷操作：申领物资、采购申请、药品总览、值日查询、制度文档
- 值日安排（实验室今日+本周排班、办公室本月）
- 日常须知（实验室/办公室分区，从数据库读取，管理端可编辑）
- 库存预警（仅管理角色可见）

### 耗材管理（使用功能）
- 物资总览：四分类，搜索
- 申领物资：**多选申领**，按分类分组展开，搜索过滤，每项设数量
  - 用途选填 + 快捷标签（课题实验、教学用、仪器维护、样品处理、安全防护、其他）
  - 可归还物品由管理员预设 `is_returnable` 属性
- 我的申领：多耗材摘要显示，点进详情
- 归还/报损：归还恢复库存，报损不恢复（中性用语，无追责）

### 药品管理（使用功能）
- 药品总览：120种药品，按编号分区（A/B/C/D），按类型分类（手机端可折叠）
- 药品信息：名称、编号、分子式、CAS号、GHS安全标签
- **取药清单**：搜索药品→加入清单→全屏查看（编号+名称，按编号排序）
  - 可保存清单到数据库（跨设备）、加载已保存清单、比对变更
- **预警优化**：确认弹窗防误触、上报人可撤回（仅 pending 状态）、一个药品只有一条有效预警
- 添加药品：编号自动生成（A/B/C/D 前缀 + 自定义）、名称输入实时显示已有相似药品

### 采购流程（统一表 purchases）
```
学生提交采购申请 → 教师审批 → 学生采购 → 补充报销信息 → 报销审批 → 入库登记
```
- **一条记录走完全流程**（审批→报销→入库）
- 11 个类别：试剂药品、实验耗材、设备配件、服装劳保、测试加工、会议培训、出版知产、办公用品、差旅交通、邮寄物流、其他
- 类别动态字段：试剂药品额外填 CAS号/浓度/纯度等，耗材设备填物品名称/规格
- 采购类型：个人/公共
- 无需入库：系统按类别自动设置，学生不可见，教师审批时可调整
- 教师/板块负责人自动审批通过
- 学生-教师对应关系：自动默认选中，可手动切换
- 附件上传（图片/PDF/Word/Excel）：申请时可上传发票，有发票+金额自动进入报销
- 报销前可修改金额和附件
- 报销金额差异 ≥50 元提醒
- 入库登记：耗材专人看非药品，药品专人看试剂药品，可编辑/删除

### 撤回功能
- 教师撤回采购审批
- 报销审批人撤回报销
- 耗材审批撤回（库存回滚）
- 管理员撤销归还（库存回滚）
- 药品补货撤回送达

### 报销统计
- 6 个 Tab：全部、试剂药品、实验耗材、服务费用、行政开支、其他
- 筛选：姓名、时间范围、金额范围、类别、采购类型（手机端可折叠）
- 汇总卡片 + 分类统计 + 导出 Excel

### 其他功能
- 制度文档：在线查看（首页+个人中心有入口）
- 值日排班：实验室每人一天/4周轮换 + 办公室每月一换（待重构为管理端可配置）
- 公告管理：发布/编辑/删除，支持附件（图片直接展示/文件下载），可选登录页展示
- 日常须知管理：实验室/办公室分区，可增删改排序
- 数据导出：耗材库存、预约记录、药品记录、采购记录（Excel 格式）

| 组件 | 技术 |

## 项目结构

```
src/
├── components/           # 通用组件（Layout, SubNav, Modal, Card...）
├── contexts/AuthContext   # 认证（角色权限helper）
├── lib/
│   ├── types.ts          # TypeScript 类型定义
│   ├── supabase.ts       # Supabase 客户端
│   ├── purchaseCategories.ts  # 采购类别配置
│   └── exportExcel.ts    # Excel 导出工具
├── pages/
│   ├── Dashboard          # 首页（角色定制，待办/公告/值日/须知）
│   ├── auth/LoginPage     # 登录页（可展示选定公告）
│   ├── supplies/          # 耗材（总览/多选申领/我的申领/归还/审批/管理/物资追踪）
│   ├── reagents/          # 药品（总览/详情/编辑/补货管理/取药清单）
│   ├── purchase-approvals/# 采购（申请/列表/审批）
│   ├── reimbursements/    # 报销（补充信息/审批/统计）
│   ├── purchases/         # 入库登记
│   ├── documents/         # 制度文档（列表/查看/编辑）
│   ├── duty/              # 值日排班
│   ├── profile/           # 个人中心（含管理入口）
│   └── admin/             # 管理后台（系统/公告/人员/日常须知/耗材管理/数据导出）
└── App.tsx                # 路由配置
```
