## 功能模块

### 登录页（无需登录即可查看）

- 公告通知（结构化展示，支持紧急/重要/普通分级）
- 值日信息（实验室今日值日 + 办公室本月值日 + 本周排班表）
- 日常须知（实验室规范 + 办公室规范，分区展示）

### 耗材管理

- **库存总览**：四分类（非一次性耗材/玻璃器皿/一次性耗材/其他），51条库存数据已导入
- **预约申请**：学生提交预约 → 管理员每周一审批 → 发放
- **借用/归还系统**：
  - 学生借用非一次性耗材（烧杯等），自动扣减库存
  - 在线归还，自动恢复库存
  - 可标记遗失
  - 管理员查看所有借用记录，按人/物筛选，手动归还

### 药品/试剂全生命周期管理

- **药品总览**：120种药品已导入，含实验室编号标签（A1~A36, B1~B46, C1~C31, D1~D2, 上-1~上-10）
- **按编号快速筛选**：A区 / B区 / C区（危化品）/ D区（冷藏）/ 上架
- **药品信息卡片**：名称、编号、分子式、CAS号、规格、浓度、纯度、厂家、存放位置
- **GHS安全标签**：9种标准图标可视化（爆炸/易燃/氧化/气体/腐蚀/剧毒/有害/健康危害/环境）
- **药品申购流程**：学生提交申购 → 负责人审批 → 记录下单 → 确认到货自动入库
- **出入库管理**：采购入库、退还入库、领用出库、废弃出库、库存调整
- **供应商管理**：名称、联系人、电话、地区
- **CAS号智能填充**：输入CAS号自动匹配已有药品信息
- **学生个人试剂台账**：
  - 查看自己使用过的所有试剂
  - 导出论文格式（按厂家分组："硝酸 (AR, 国药集团)"）
  - 导出Excel完整记录

### 制度文档中心

- 在线查看/编辑所有规章制度（4篇已录入）
- 已录入：危化品管理制度、组会管理细则、物资管理说明、日常行为准则

### 值日排班

- **实验室值日**：每人一天（周一~周五），每4周轮换一次（每人往后挪一天）
- **办公室值日**：每月一换，一人负责一整月
- 登录页自动显示今日值日人和本周排班

### 其他功能

- **仪器设备管理**：设备列表、负责人、状态跟踪、故障上报
- **组会管理**：日程安排、汇报顺序、Progress Report 提交
- **报销申请**：提交申请、审批流程、记录查询

### 管理后台（中文界面，不需要编程）

- **公告管理**：发布/编辑/删除公告，支持优先级分级，实时预览
- **人员管理**：直接创建账号（填邮箱+密码+姓名+角色）、重置密码、删除账号
- **权限分配与交接**：选择板块 → 选择新负责人 → 一键交接
- **耗材管理**：添加/编辑/删除耗材，批量管理库存
- **借用管理**：查看所有借用记录，按人/物筛选，手动归还
- **供应商管理**：管理试剂供应商信息
- **数据导出**：一键导出耗材库存/预约记录/危化品记录/报销记录为CSV

### 用户认证

- 邮箱+密码登录（邮箱仅作账号名，无需真实邮箱，不发送邮件）
- 管理员直接创建账号，无需注册流程
- 5种角色：超级管理员/管理员/板块负责人/教师/学生
- 可授权其他人管理"人员管理"模块
- 自动清除无效 session，6秒超时保护

## 技术栈

- **前端**：React + TypeScript + TailwindCSS v4 + Vite
- **后端/数据库**：Supabase（PostgreSQL + Auth + RLS）
- **部署**：Cloudflare Pages

## 部署更新


## 数据库

数据库包含 19 张表：

| 表名 | 用途 |
|------|------|
| `profiles` | 用户信息（扩展 auth.users） |
| `announcements` | 公告 |
| `documents` | 制度文档 |
| `supply_categories` | 耗材分类（4类） |
| `supplies` | 耗材库存（51条数据） |
| `supply_reservations` | 耗材预约 |
| `supply_borrowings` | 耗材借用/归还记录 |
| `chemicals` | 药品/试剂（120种，含编号、GHS标签、厂家等） |
| `chemical_usage_logs` | 药品使用记录 |
| `chemical_purchases` | 药品采购（旧版） |
| `suppliers` | 供应商 |
| `reagent_stock_movements` | 药品出入库记录 |
| `reagent_purchase_requests` | 药品申购单 |
| `duty_roster` | 值日排班 |
| `equipment` | 仪器设备 |
| `meetings` | 组会 |
| `meeting_reports` | Progress Report |
| `reimbursements` | 报销申请 |
| `purchase_logs` | 物资购买登记 |

### 数据库自定义函数

| 函数 | 用途 |
|------|------|
| `create_lab_user()` | 管理员创建账号（含identity记录） |
| `reset_user_password()` | 管理员重置密码 |
| `delete_lab_user()` | 管理员删除账号 |
| `handle_new_user()` | 注册时自动创建profile |
| `is_admin_or_above()` | RLS权限检查 |
| `manages_module()` | RLS模块权限检查 |

## 项目结构

```
src/
├── components/          # 通用 UI 组件（8个）
│   ├── Layout.tsx       # 主布局（侧边栏 + 底部导航 + 用户菜单）
│   ├── ProtectedRoute.tsx
│   ├── Modal.tsx, Card.tsx, PageHeader.tsx, StatusBadge.tsx, ...
├── contexts/
│   └── AuthContext.tsx   # 用户认证（自动清除无效session）
├── lib/
│   ├── supabase.ts      # Supabase 客户端
│   └── types.ts         # TypeScript 类型定义
├── pages/
│   ├── Dashboard.tsx     # 首页总览
│   ├── admin/           # 管理后台（6个页面）
│   ├── auth/            # 登录页（含公告/值日/须知）
│   ├── chemicals/       # 危化品管理（旧版，4个页面）
│   ├── reagents/        # 药品全生命周期管理（7个页面）
│   ├── documents/       # 制度文档（3个页面）
│   ├── duty/            # 值日排班 & 仪器设备（2个页面）
│   ├── meetings/        # 组会管理（2个页面）
│   ├── profile/         # 个人中心
│   ├── reimbursements/  # 报销管理（3个页面）
│   └── supplies/        # 耗材管理（7个页面，含借用/归还）
└── App.tsx              # 路由配置
```
